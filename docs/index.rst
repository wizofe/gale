Getting Started
---------------

.. code-block:: bash

   # Install CPU-only version
   pip install -e .

   # Run analysis
   neurogale --n_voxels 1000 --n_timepoints 200 --cpu-only

   # Check GPU availability
   neurogale --show-gpu-info

For detailed installation instructions, see :doc:`installation`.
